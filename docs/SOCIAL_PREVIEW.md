# Social Preview Image Guide

GitHub recommends a social preview image of **1280x640 pixels** for optimal display when your repository is shared on social media.

## Design Recommendations

### Content
- Project name: **Smart XDebug MCP**
- Tagline: **Let AI debug your PHP code**
- Visual: Show the architecture flow or a debugging example

### Layout Example
```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    Smart XDebug MCP                          │
│                                                              │
│              Let AI debug your PHP code                      │
│                                                              │
│     ┌─────────┐     ┌─────────┐     ┌─────────┐             │
│     │ Claude  │ ──► │   MCP   │ ──► │ XDebug  │             │
│     └─────────┘     └─────────┘     └─────────┘             │
│                                                              │
│     Runtime debugging • Breakpoints • Variable inspection    │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### Style Guidelines
- Keep text centered and minimal
- Use high contrast colors
- Avoid more than 20% text coverage (Facebook penalty)
- Include visual elements (icons, diagrams)

## Tools to Create

1. **Figma** (free): https://figma.com
2. **Canva** (free): https://canva.com - has GitHub social preview templates
3. **GIMP** (free, open source): https://gimp.org

## Setting the Image

1. Go to repository Settings
2. Under "Social preview", click "Edit"
3. Upload your 1280x640 image
4. Save changes

## Verify

Test your social preview:
- Facebook: https://developers.facebook.com/tools/debug/
- Twitter: https://cards-dev.twitter.com/validator
- LinkedIn: https://www.linkedin.com/post-inspector/
